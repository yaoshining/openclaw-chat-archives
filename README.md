# OpenClaw Chat Archives

Auto-generated chat exports from OpenClaw with full Markdown rendering.

## Usage

Chat messages exported from OpenClaw appear here when they exceed the length threshold. Each file contains:

- Title
- Export timestamp  
- Full chat content
- Auto-generation note

## File Format

- **Naming**: `YYYY-MM-DD-HHMMSS.md`
- **Location**: `/messages/` directory
- **Rendering**: Native GitHub Markdown

## View Files

Browse files in the [`messages/`](messages/) directory.
